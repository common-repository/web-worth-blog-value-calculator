== Description ==

Blog or Website Value Calculator is a plug-in that gives you a nice tidy widget to display your blog's current estimated value in dollars. This information is displayed to you and your visitors to demonstrate just how much hard work you have put into your blog - Why not show this cool statistic off? This widget is dynamically updated when the sites value is updated at Web Worth. The site must be valued at Web Worth before it will show a value in dollars otherwise you will just see a VALUATE NOW warning.

== Installation ==

This section describes how to install the plugin and get it working.

1. Download the plugin, extract and copy to your ../wp-content/plugins directory on your webserver

2. Enable the plugin on the plugins page in the wordpress admin tool

3. Once enabled if you click widgets under apperance then there will be an extra widget call blog value

4. Add this to the panel of your choice

5. Once added if you viset your blog and you will see a nice little graphic showing you and your visitors your blog's value. If you see VALUATE NOW - this means your blog has not yet been valuated. You need to click VALUATE NOW and it will valuate your blog once complete, if you refresh your blog's page it will show the value.

6. Done!
